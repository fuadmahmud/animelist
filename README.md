# Bootstraping using vite.js

## 1. Technology used
- [vite](https://vitejs.dev/)
- [mui](https://mui.com/)
- [@emotion](https://emotion.sh/docs/introduction)
- [@apollo/client](https://www.apollographql.com/docs/react/)
- [react-router-dom](https://reactrouter.com/en/main)

## 2. How To Run

```
- npm i
- npm run dev
- open localhost:3000
```
